# **App Name**: Violet Bites

## Core Features:

- User Authentication: Secure login page with options for user and admin roles.
- Menu Display: Display of meals and snacks menus with food items.
- Add to Cart Functionality: Option to add selected food items to a cart.
- Order Summary and Checkout: Display cart items, calculate total price including convenience fee, and provide payment options.
- Payment Processing: Link user payments to the canteen owner's account.
- Order Management: Admin interface to view and manage incoming orders.
- Reporting and Analytics: Admin dashboard with canteen details, payment statistics, and sales analysis.

## Style Guidelines:

- Primary color: Deep violet (#673AB7) to match the overall theme.
- Background color: Light lavender (#E1D9ED), a very light tint of violet.
- Accent color: Plum (#9C27B0), for highlights and active elements, providing contrast.
- Body and headline font: 'PT Sans', sans-serif, to provide a balance of modernity and readability.
- Use clear and consistent icons to represent menu categories and actions.
- Clean and intuitive layout with clear visual hierarchy to enhance user experience.
- Subtle animations and transitions to provide feedback and improve engagement.
